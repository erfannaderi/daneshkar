# README.md

## Installation


1. Install the required packages using pip:

   ```
   pip install -r requirements.txt
   ```

## Usage

To access https://google.com, follow these steps:

1. Open the terminal.

2. Navigate to the project directory:

   ```
   cd place you want to run it
   ```

3. Run the command:

   ```
   python main.py -u https://goole.com -t 15 -r
   ```

   This command will execute the Python script `main.py` which accesses the URL https://google.com.
    or just run it ;)

4. The script will retrieve the content of the webpage and display it in the terminal.

   ````
   Page content: 
   ```

   Replace  with the actual content retrieved from https://google.com.

That's it! You have successfully installed the requirements and executed the command to access https://google.com. Feel free to modify the code or explore other functionalities as needed.
